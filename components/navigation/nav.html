Navigation

Basic Main Navigation

The basic main navigation bar is great when you have only a few navigation items.

To indicate a current item, add class main-nav__link--current.

Item Icons

To add an icon to a navigation item, use class main-nav__icon, followed by the class name of the icon.

Dropdown Menu

Always use both hover and click to trigger dropdown menus. Although it is possible to detect whether a device supports touch screen or not, it is safer to always include click support, and never assume a device is exclusively touch or the other.
